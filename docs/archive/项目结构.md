## 根目录一览

| 路径 | 用途 | 是否上传服务器 |
|------|------|----------------|
| `src/` | 前端页面与组件 | 否（上传 build 后的 `dist/`） |
| `server/` | 后端 API 与业务逻辑 | 否（上传 build 后的 `dist-server/`） |
| `dist/` | 前端构建产物 | **是** |
| `dist-server/` | 后端构建产物 | **是** |
| `public/` | 静态资源 | **是** |
| `data/` | 本地 SQLite（约 200MB+） | **否**（生产用 Docker 卷） |
| `deploy/` | Nginx 配置示例 | 可选 |
| `docs/` | 文档 | 可选 |
| `scripts/` | **部署打包脚本** | 否 |
| `tools/` | **本地实验**（含 alipay + `.venv`，与看板无关） | **否** |
| `secrets/` | 私钥等敏感文件 | **否** |
| `release/` | 打好的 `.tgz` 部署包 | 否 |
| `node_modules/` | npm 依赖 | **否** |

---

## 配置文件

| 文件 | 说明 |
|------|------|
| `.env` | 看板 JWT 等（Mac 可选，**服务器必须有**） |
| `.env.example` | 看板环境变量模板 |
| `docker-compose.yml` | Docker 启动（使用 `Dockerfile.prod`） |
| `Dockerfile.prod` | **生产用**：直接用 dist，不在服务器编译 |
| `Dockerfile` | 完整编译版（本地调试 Docker 时用） |
| `vite.config.ts` | 前端构建配置 |

**不要**把 `tools/alipay/.env` 或 `tools/alipay/.venv/` 当成看板配置上传。

## 常用命令（Mac）

```bash
cd /Users/Eric/projects/sales-data-agent

npm run dev              # 本地开发
npm run build            # 生成 dist + dist-server

bash scripts/deploy-pack.sh      # 全量部署包 → release/sales-agent-日期.tgz
bash scripts/deploy-frontend.sh  # 仅前端 → release/sales-dist-日期.tgz
```

## 服务器部署（OrcaTerm）

```bash
cd /opt/sales-data-agent
# 解压前备份 .env
cp .env /tmp/.env.backup
sudo tar xzf ~/sales-agent-xxx.tgz -C /opt/sales-data-agent
cp /tmp/.env.backup .env
sudo docker compose up -d --build
```

仅更新前端：

```bash
sudo tar xzf ~/sales-dist-xxx.tgz -C /opt/sales-data-agent
sudo docker compose up -d --build
```

（`restart` 不会更新镜像里的 `dist`，必须 `--build`。）
