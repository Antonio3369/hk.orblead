## 三、2026-07-05 功能纪要（商户洞察看板）

### 3.1 指标定义（已定稿）

| 指标 | 定义 |
|------|------|
| **本月 MTD** | 本月 1 日 00:00 至昨日 23:59 交易额 |
| **日均环比** | 本月日均 vs 上月日均（%） |
| **新沉默** | 归属该销售；**上月完整自然月 sum > 0**；**本月 MTD sum = 0** |
| **下跌中** | 本月 MTD > 0；且日均环比 **< -阈值**（默认 **10%**，admin 可调） |
| **上涨** | 本月 MTD > 0；且日均环比 > 0 |
| **预警跟进** | 现有 alerts：`week` / `month` 维度、未跟进的预警（阈值默认 30%，与「下跌中」口径不同） |

**关系**：「下跌中」比「交易预警」更早发现趋势下滑；「新沉默」表示已停交易；二者互补不重复。

### 3.2 各角色界面

| 角色 | 入口 | 行为 |
|------|------|------|
| **admin** | 飞虎队 | 默认 **上月交易** 排序；可切 新沉默 / 下跌中 / 上涨 / 预警跟进；点销售进详情 |
| **leader** | 我的团队 | 同 admin，数据范围为本团队 |
| **leader** | 我的商户 | 有 insight 筛选 + 状态列；**首页无「摘要」卡** |
| **leader** | 手机业务卡包 | 第 4 格：**團隊**（非摘要） |
| **sales** | 我的商户 | insight 筛选 + **状态列**；**保留**刷卡/扫码额度 % 列；默认 **上月交易** |
| **sales** | 首页业务卡包 | 第 4 格：**摘要** → 点进看 MTD/环比/计数，再跳转我的商户对应筛选 |
| **admin** | 全部商户 | **不变**：表头排序，无 insight 按钮 |
| **admin** | 后台管理 | **下跌中判定（日均环比）** 阈值设置 |

### 3.3 预警跟进差异（SOP）

- **管理者**进销售详情 →「预警跟进」：商户列表 **+** 下方预警明细与跟进记录
- **销售**在「我的商户」→「预警跟进」：**仅筛选**有未跟进预警的商户；详细跟进请去 **「交易预警」** 页
