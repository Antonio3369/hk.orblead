## 三、2026-07-05 功能纪要（商户洞察看板）

### 3.1 指标定义（已定稿）

| 指标 | 定义 |
|------|------|
| **本月 MTD** | 本月 1 日 00:00 至昨日 23:59 交易额 |
| **日均环比** | 本月日均 vs 上月日均（%） |
| **新沉默** | 归属该销售；**上月完整自然月 sum > 0**；**本月 MTD sum = 0** |
| **下跌中** | 本月 MTD > 0；且日均环比 **< -阈值**（默认 **10%**，admin 可调） |
| **上涨** | 本月 MTD > 0；且日均环比 > 0 |
| **预警跟进** | 现有 alerts：`week` / `month` 维度、未跟进的预警（阈值默认 30%，与「下跌中」口径不同） |

**关系**：「下跌中」比「交易预警」更早发现趋势下滑；「新沉默」表示已停交易；二者互补不重复。

### 3.2 各角色界面

> **2026-07-11 起**主入口改为 **左侧侧栏**；Leader 工作台 = 本人，团队见「我的團隊」。详见 [`立得香港看板-AI助手交接指南.md`](./立得香港看板-AI助手交接指南.md) §五。

| 角色 | 入口 | 行为 |
|------|------|------|
| **admin** | 工作台 | 全机构图表、飞虎队/商户票房榜 |
| **admin** | 飛虎隊（侧栏） | 默认 **上月交易** 排序；可切 insight 筛选；点销售进详情 |
| **leader** | 工作台 | **本人**商户 MTD、图表、统计（scope 文案明确） |
| **leader** | 我的團隊（侧栏） | 团队合计、成员排名、团队票房；同 admin 飞虎队交互 |
| **leader** | 我的與團隊商戶 | insight 筛选 + 状态列；列表含团队范围 |
| **sales** | 工作台 | MTD Hero + 图表 + insight 跳转（无独立「摘要」侧栏项） |
| **sales** | 我的商戶 | insight 筛选 + **状态列**；**保留**刷卡/扫码额度 % 列 |
| **admin** | 全部商戶 | 表头排序，无 insight 按钮 |
| **admin** | 後臺管理 | 阈值设置、导入、**改登入名**、**開發者視圖** |

### 3.3 预警跟进差异（SOP）

- **管理者**进销售详情 →「预警跟进」：商户列表 **+** 下方预警明细与跟进记录
- **销售**在「我的商户」→「预警跟进」：**仅筛选**有未跟进预警的商户；详细跟进请去 **「交易预警」** 页

#### 2026-07-18 · 导入性能 + 上传体验（✅ 已发版）

| 项 | 内容 |
|----|------|
| 详细说明 | 交接指南 §10.5、§11.2 |
| 生产验证 | 2026-07-18 下午；22KB 重复 append 数秒内完成，整站不再卡死 |

**变更要点**

- append 重复导入：**批量 IN 查订单号**、已存在订单跳过 upsert/INSERT、**批量 UPDATE** 补空卡歸屬地。
- 纯重复/只补列：**不重算全库预警**；有新交易时按涉及商户重算，且在 HTTP 响应后异步执行。
- Nginx `/api/import` 超时 600s（优化后通常不需等满）。
- 后台上传：小文件也有进度回调；多文件进度只增不减；失败有明确提示。

**勿踩**：导入慢/整站卡 **不是** 腾讯云带宽或 CPU 不够，勿盲目升配。
